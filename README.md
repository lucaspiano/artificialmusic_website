# Artificial Music

Pequeno template React + Vite com TailwindCSS para o site "Artificial Music".

Pré-requisitos
- Node.js 18+ (recomendado)

Instalação
```bash
npm install
```

Executar em modo desenvolvimento
```bash
npm run dev
```

Build para produção
```bash
npm run build
npm run preview
```

Notas rápidas
- Arquivos principais: `index.html`, `package.json`, `tailwind.config.js`, `postcss.config.js`.
- Código-fonte em: `src/` (veja `src/main.jsx` e `src/App.jsx`).
- Páginas: `src/pages/*` (Home, About, Products, Research, Future, Contact).
- Componentes: `src/components/*` (Navbar, Footer).
- Para alterar o e‑mail de contato edite `src/pages/Contact.jsx`.

Se precisar, posso:
- Rodar `npm install` e iniciar o servidor local aqui.
- Adicionar um `.gitignore` ou configurar deploy (Netlify/Vercel).
